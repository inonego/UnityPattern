# 유니티 패턴 라이브러리

- 유니티에서 자주 사용하는 디자인 패턴과 유틸리티 기능들을 구현해놓은 라이브러리입니다.
- 프로젝트 초기 세팅부터 게임 로직 구현까지 특정 프로젝트에 의존하지 않고 적용할 수 있도록 구현하는 것이 목표입니다.

## 📂 프로젝트 구조 및 주요 기능

### 🛠️ [게임 코어 (Game Core)](./게임%20코어)
- **[Bootstrapper](./게임%20코어/Bootstrapper)**: 게임 시작 시 필수 모듈을 순차적으로 초기화하는 시스템입니다.

### 🎮 [게임 로직 (Game Logic)](./게임%20로직)
- **[레벨 시스템](./게임%20로직/레벨)**: 단순한 레벨부터 경험치 곡선 및 레벨 업 로직을 관리하도록 합니다.
- **[보드 로직](./게임%20로직/보드)**: 2D/3D 그리드 기반의 공간 데이터 및 이동 규칙을 처리합니다.
- **[엔티티](./게임%20로직/엔티티)**: 플레이어, 몹 등을 구현할때 아이디 관리, 스폰 / 디스폰 기능을을 사용할 수 있습니다.
- **[스폰](./게임%20로직/스폰)**: 엔티티나 그와 비슷한 객체에 대해 스폰 로직을 관리합니다.
- **[체력 관리 (HP)](./게임%20로직/체력)**: HP에 대한 구현체로 웬만한 게임에 바로 적용 가능합니다.
- **[컨트롤러](./게임%20로직/컨트롤러)**: 바닥 감지(Ground Checker), 점프 제어 등 이동 유틸리티입니다.

### 🖼️ [게임 UI (Game UI)](./게임%20UI)
- **[Bar UI](./게임%20UI/Bar%20UI)**: HP/MP 바 등 게이지 형태의 UI를 쉽게 구현할 수 있는 컴포넌트입니다.
- **[Drag & Drop](./게임%20UI/Drag_Drop%20UI)**: 아이템 슬롯 등 드래그 앤 드롭 기능을 제공합니다.
- **[SceneFader](./게임%20UI/SceneFader)**: 화면을 부드럽게 페이드 인/아웃 시킬 수 있습니다.

### 🔧 [유틸리티 (Utility)](./유틸리티)
- **[오브젝트 풀링](./유틸리티/게임%20오브젝트/오브젝트%20풀링)**: 빈번한 인스턴스 생성/삭제를 최적화하는 풀링 시스템입니다.
- **[데이터 유틸리티](./유틸리티/데이터)**: 직렬화 가능한 컬렉션, 데이터 테이블 등을 포함합니다.
- **[싱글톤 패턴](./유틸리티/싱글톤)**: `Singleton<T>`, `MonoSingleton<T>` 등 다양한 싱글톤 구현체를 제공합니다.
- **[타이머](./유틸리티/타이머)**: 쿨타임, 지속 시간 측정 등에 사용할 수 있는 타이머입니다.
- **[로깅 시스템](./유틸리티/로그)**: 확장된 기능을 제공하는 커스텀 디버그 로거입니다.
- **[이벤트 유틸리티](./유틸리티/이벤트)**: 데이터 변경 감지(Dirty Flag) 및 각종 델리게이트 헬퍼입니다.
